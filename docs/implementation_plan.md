# ZahnerFlow 功能储备计划

---

## 一、核心工作流功能

### 1. 多工作流串行执行
批量运行多个工作流，按队列顺序依次执行。

**技术方案**：
- `ExecutionQueueService` - 队列管理
- 前端 `WorkflowQueuePanel` - 队列管理UI

---

## 二、系统级功能

### 2. 条件邮件通知
根据工作流执行结果自动发送邮件。

**触发条件**：
- 工作流完成时
- 工作流失败时
- 执行时间超过阈值
- 指定节点完成后

**配置项**：收件人、主题模板、内容模板、附件选项

---

### 3. 错误检测与自动处理
监控设备异常并触发自动响应。

**检测条件**：
| 条件 | 描述 | 默认阈值 |
|------|------|----------|
| EIS 无响应 | 设备通信超时 | 30 分钟 |
| 无新数据 | 数据流中断 | 5 分钟 |
| 数值越界 | 参数超出范围 | 自定义 |
| 连接丢失 | 设备断开 | 即时 |

**自动响应**：停止执行、重试、发送告警、跳过当前节点

---

### 4. 文件自动命名
测量文件根据模板自动重命名。

**变量支持**：`{date}`, `{time}`, `{user}`, `{project}`, `{workflow}`, `{node}`

**示例**：`{project}_{date}_{node}.csv` → `MyProject_20231214_EIS.csv`

---

### 5. 报警系统
多级别、多渠道的告警通知。

**级别**：INFO / WARNING / ERROR / CRITICAL

**渠道**：WebSocket弹窗、邮件、声音提示

---

## 三、扩展功能建议

以下是可能对用户有帮助的额外功能：

### 6. 执行历史与统计
- 记录每次工作流执行的详细日志
- 统计成功率、平均执行时间、设备使用率
- 可视化图表展示历史趋势

---

### 7. 工作流模板库
- 保存常用工作流为模板
- 模板分类管理（按实验类型、设备类型）
- 一键从模板创建新工作流
- 模板共享导入/导出

---

### 8. 参数预设管理
- 保存常用的节点参数组合
- 如"标准 EIS 测量"预设：频率范围、点数、振幅等
- 快速应用预设到节点

---

### 9. 批量数据导出
- 选择多个历史执行结果
- 统一导出为 CSV / Excel / JSON
- 自定义导出字段和格式
- 支持数据打包 ZIP 下载

---

### 10. 设备状态仪表盘
- 实时显示所有设备连接状态
- 温度、气体流量等实时读数
- 设备健康指标（响应时间、错误计数）
- 设备使用时长统计

---

### 11. 实验日志系统
- 自动记录关键事件：启动、停止、错误、参数变更
- 支持手动添加备注
- 日志搜索和过滤
- 导出实验报告

---

### 12. 工作流备份与恢复
- 自动定期备份工作流配置
- 版本历史管理
- 一键恢复到历史版本
- 导出/导入工作流 JSON

---

### 13. 数据校验规则
- 为测量结果设置预期范围
- 自动标记异常数据点
- 数据质量评分
- 异常时触发告警或暂停

---

### 14. 执行时间预估
- 根据节点参数计算预计执行时间
- 显示工作流总时长预估
- 串行队列完成时间预估
- 历史执行时间参考

---

### 15. 快捷键系统
- 常用操作快捷键：运行(Ctrl+Enter)、停止(Esc)、保存(Ctrl+S)
- 可自定义快捷键映射
- 快捷键提示面板

---

## 四、功能优先级矩阵

```
        高价值
           │
     ┌─────┼─────┐
     │  3  │  1  │  ← 优先实施
     │错误 │多工 │
     │检测 │作流 │
─────┼─────┼─────┼───── 低工作量 ← → 高工作量
     │  5  │  6  │
     │报警 │历史 │
     │系统 │统计 │
     └─────┼─────┘
           │
        低价值
```

**第一优先级**：1, 3, 5 (核心可靠性)
**第二优先级**：2, 4, 14 (用户体验)
**第三优先级**：6, 7, 8, 11 (效率提升)
**第四优先级**：9, 10, 12, 13, 15 (锦上添花)

## 五、技术实现要点

| 功能 | 后端改动 | 前端改动 | 数据库 |
|------|----------|----------|--------|
| 多工作流串行 | `ExecutionQueueService` | `QueuePanel` | `execution_queue` 表 |
| 条件邮件 | 增强 `EmailService` | 设置面板 | 邮件配置表 |
| 错误检测 | `ErrorDetectorService` | 状态显示 | 无 |
| 文件命名 | `FilesService` 增强 | 设置面板 | 无 |
| 报警系统 | `AlertService` | 告警面板 | 告警规则表 |
| 执行历史 | 增强日志记录 | 历史面板 | 增强现有表 |
| 模板库 | `TemplateService` | 模板面板 | `workflow_templates` 表 |
| 参数预设 | `PresetService` | 预设选择器 | `parameter_presets` 表 |
