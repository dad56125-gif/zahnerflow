# ZAHNERFLOW 数据库模块

这是一个创新的数据库实现方案，专为 ZAHNERFLOW 电化学工作流管理系统设计。

## 特性

- 🚀 **高性能**: 使用 TimescaleDB 处理时序数据
- 🔄 **灵活扩展**: 支持多种数据库（PostgreSQL、SQLite）
- 📊 **智能缓存**: Redis 多层缓存策略
- 🔍 **优化查询**: 复合索引和查询优化
- 🛡️ **数据安全**: 事务处理和软删除
- 📈 **实时监控**: 性能指标和慢查询跟踪

## 快速开始

### 1. 安装

```bash
npm install
```

### 2. 配置

复制环境变量文件：

```bash
cp .env.example .env
```

### 3. 运行

开发环境：
```bash
npm run start:dev
```

生产环境：
```bash
npm run build
npm run start:prod
```

## 创新点

### 1. 时序数据优化
使用 TimescaleDB 超表技术，自动分区和压缩，大幅提升时序查询性能。

### 2. 智能缓存策略
- 多级缓存（Redis + 应用缓存）
- 智能失效机制
- 缓存命中率监控

### 3. 灵活的数据模型
- JSONB 存储动态结构
- 支持多种实验类型
- 扩展性强

### 4. 实时性能监控
- 慢查询检测
- 连接池监控
- 自动优化建议
